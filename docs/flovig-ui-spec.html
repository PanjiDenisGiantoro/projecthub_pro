Foundation 03

Spacing & Radius

Badge digunakan untuk status, label kategori, dan hitungan notifikasi. Selalu gunakan warna semantik yang tepat — jangan menggunakan warna acak.

Pattern 01

CTA Sections

Tiga varian CTA: Full (halaman landing, akhir section), Minimal (in-app prompt), Banner (top-of-page notification). Setiap CTA HARUS punya trust signal di bawah tombol.

Full CTA — Landing Page
🚀 Mulai sekarang

Siap menyederhanakan operasi tim Anda?

Coba Flovig gratis 14 hari. Tanpa kartu kredit, tanpa komitmen.

✓ Tanpa kartu kredit ✓ Setup < 5 menit ✓ Cancel kapan saja ✓ Data aman
Minimal CTA — In-App

Lengkapi konfigurasi HRIS Anda

Tambahkan departemen untuk mulai mengelola struktur organisasi

Banner CTA — Top Notification

3 pengajuan cuti menunggu persetujuan

Ditambahkan dalam 24 jam terakhir

CTA Rules: (1) Satu primary action per CTA — jangan dua tombol primary bersebelahan. (2) Selalu tambahkan trust signal (tanpa CC, cancel kapan saja, dll) di bawah CTA utama. (3) Copy CTA harus action-verb: "Mulai", "Lihat", "Review" — hindari "Klik di sini" atau "Submit".

CSS Design Tokens

Semua token wajib diimplementasi sebagai CSS Custom Properties di resources/css/app.css. Tailwind config harus extend dari token ini.

/* ═══════════════════════════════════ FLOVIG DESIGN TOKENS v1.0 resources/css/app.css ═══════════════════════════════════ */ :root { /* ── Lavender Brand ── */ --lav-50: #faf5ff; --lav-100: #f3e8ff; --lav-200: #e9d5ff; --lav-300: #d8b4fe; --lav-400: #c4b5fd; /* Active nav text */ --lav-500: #a78bfa; /* Icon accent, links */ --lav-600: #8b5cf6; /* Hover state */ --lav-700: #7c3aed; /* ★ PRIMARY — CTA, active */ --lav-800: #6d28d9; /* CTA gradient end */ --lav-900: #5b21b6; --lav-950: #2e1065; /* Dark sidebar bg */ /* ── Surfaces (Dark Mode) ── */ --surface-page: #09061a; /* Page background */ --surface-sidebar: #120a28; /* Sidebar */ --surface-card: rgba(255,255,255,0.03); --surface-hover: rgba(167,139,250,0.07); --border-default: rgba(167,139,250,0.10); --border-subtle: rgba(167,139,250,0.06); /* ── Surfaces (Light Mode) ── */ --surface-page-lt: #f4f2fb; --surface-sidebar-lt: #ffffff; --surface-card-lt: #ffffff; --border-default-lt: #ede9fe; /* ── Typography ── */ --font-display: 'Plus Jakarta Sans', sans-serif; --font-body: 'Inter', sans-serif; --font-mono: 'JetBrains Mono', monospace; /* ── Semantic ── */ --color-success: #10b981; --color-warning: #f59e0b; --color-danger: #ef4444; --color-info: #3b82f6; /* ── Radius ── */ --r-sm: 6px; /* badge, tag */ --r-md: 9px; /* input, button-sm */ --r-lg: 10px; /* button default, nav item */ --r-xl: 14px; /* card, stat card */ --r-2xl: 20px; /* section CTA, modal */ /* ── Shadows ── */ --shadow-btn: 0 4px 16px rgba(109,40,217,0.4); --shadow-card: 0 8px 32px rgba(109,40,217,0.12); --shadow-hover:0 12px 32px rgba(109,40,217,0.18); }
Tailwind extend: Tambahkan colors.lav dan fontFamily.display ke tailwind.config.js agar class seperti text-lav-700, bg-lav-100, dan font-display tersedia di Blade templates.
